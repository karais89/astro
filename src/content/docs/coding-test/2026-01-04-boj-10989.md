# BOJ 10989 수 정렬하기 3

## 문제

N개의 수가 주어졌을 때, 이를 오름차순으로 정렬하는 프로그램을 작성하시오.

## 입력

- 첫째 줄: 수의 개수 N (1 ≤ N ≤ 10,000,000)
- 둘째 줄부터 N개 줄: 수 (1 ≤ 수 ≤ 10,000)

## 출력

- N개의 수를 오름차순으로 한 줄에 하나씩 출력

## 접근

메모리 제한이 8MB로 매우 작기 때문에 일반적인 정렬 알고리즘으로는 해결할 수 없다. 수의 범위가 1~10,000으로 제한되어 있으므로 카운팅 정렬을 사용해야 한다.

카운팅 정렬은 각 수가 등장한 횟수를 저장한 후, 빈도만큼 출력하는 방식으로 O(N + K)의 시간 복잡도를 가진다. (K는 수의 최대값)

## 풀이 코드 (C#)

```csharp
using System;
using System.IO;

using var sr = new StreamReader(Console.OpenStandardInput());
using var sw = new StreamWriter(Console.OpenStandardOutput());

int n = int.Parse(sr.ReadLine());
int[] count = new int[10001];

for (int i = 0; i < n; i++)
{
    int num = int.Parse(sr.ReadLine());
    count[num]++;
}

for (int i = 1; i <= 10000; i++)
{
    for (int j = 0; j < count[i]; j++)
    {
        sw.WriteLine(i);
    }
}
```

## 메모

- 핵심 아이디어: 메모리 제한이 작으므로 카운팅 정렬 사용
- 수의 범위가 제한되어 있어 카운팅 정렬 가능
- Console.ReadLine/WriteLine 대신 StreamReader/StreamWriter 사용으로 시간 최적화
