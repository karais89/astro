# BOJ 1436 영화감독 숌

## 문제

666은 종말을 나타내는 숫자이다. 따라서 많은 블록버스터 영화에서는 666이 들어간 제목을 짓곤 한다.

영화감독 숌은 세상의 종말을 다룬 영화를 많이 만들었다. 숌은 영화의 제목을 정하는 독특한 규칙을 가지고 있다.

종말의 숫자란 어떤 수에 6이 적어도 3개 이상 연속으로 들어가는 수를 말한다. 제일 작은 종말의 숫자는 666이고, 그 다음은 1666, 2666, 3666, .... 과 같다.

N번째 종말의 숫자를 출력하는 프로그램을 작성하시오.

## 입력

- 첫째 줄: N (1 ≤ N ≤ 10,000)

## 출력

- N번째 종말의 숫자를 출력

## 접근

이 문제는 브루트포스로 해결할 수 있다.

### 알고리즘

1. 666부터 시작하여 숫자를 하나씩 증가시킨다
2. 각 숫자를 문자열로 변환하여 "666"이 포함되어 있는지 확인한다
3. "666"이 포함되어 있으면 카운터를 1 증가시킨다
4. 카운터가 N에 도달하면 해당 숫자를 출력하고 종료한다

### 시간 복잡도

- N번째 종말의 숫자는 대략 N × 1000 이하이므로, N이 10,000일 때 대략 10,000,000번 이하의 연산
- 각 숫자마다 문자열 변환 및 Contains 연산이 상수 시간이므로 충분히 빠름

## 풀이 코드 (C#)

```csharp
using System;

// 666, 1666, 2666, 3666, 4666, 5666, 6660, 6666
int findNumber = int.Parse(Console.ReadLine()!);
int i = 666;
int number = 0;

while (number < findNumber)
{
    if (i.ToString().Contains("666"))
    {
        number++;
        if (number == findNumber)
        {
            Console.WriteLine(i);
            break;
        }
    }
    
    i++;
}
```

## 메모

- 핵심 아이디어: 666부터 순회하며 "666"을 포함하는 숫자를 카운트
- 브루트포스 방식으로 충분히 해결 가능한 문제
- `Contains` 메서드를 사용하여 부분 문자열 검색
- 패턴을 찾기보다는 단순하게 모든 경우를 탐색하는 것이 더 간단함
