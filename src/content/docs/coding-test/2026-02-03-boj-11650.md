# BOJ 11650 좌표 정렬하기

## 문제

2차원 평면 위의 점 N개가 주어진다. 좌표를 x좌표가 증가하는 순으로, x좌표가 같으면 y좌표가 증가하는 순서로 정렬한 다음 출력하는 프로그램을 작성하시오.

## 입력

- 첫째 줄: 점의 개수 N (1 ≤ N ≤ 100,000)
- 다음 N개 줄: i번 점의 위치 xi와 yi (-100,000 ≤ xi, yi ≤ 100,000)
  - 좌표는 항상 정수이고, 위치가 같은 두 점은 없다

## 출력

- 첫째 줄부터 N개의 줄에 점을 정렬한 결과를 출력

## 접근

기본적인 정렬 문제로, 정렬 기준을 명확히 설정하는 것이 핵심이다.

### 정렬 기준

1. **1순위**: x좌표 오름차순
2. **2순위**: x좌표가 같을 경우 y좌표 오름차순

### 알고리즘

1. N개의 좌표를 입력받아 리스트에 저장
2. 커스텀 비교 함수를 사용하여 정렬
   - x좌표를 먼저 비교
   - x좌표가 같으면 y좌표 비교
3. 정렬된 결과를 출력

### 주의사항

- C#의 `Sort` 메서드에 람다 표현식으로 비교 로직 전달
- Comparer는 음수(-1), 0, 양수(1)를 반환하여 정렬 순서 결정
- x좌표가 다르면 x좌표 비교 결과를, 같으면 y좌표 비교 결과를 반환

## 풀이 코드 (C#)

```csharp
using System;
using System.Collections.Generic;

int n = int.Parse(Console.ReadLine()!);
List<(int x, int y)> points = new List<(int, int)>();

for (int i = 0; i < n; i++)
{
    int[] values = Array.ConvertAll(Console.ReadLine()!.Split(), int.Parse);
    points.Add((values[0], values[1]));
}

points.Sort((a, b) =>
{
    if (a.x > b.x)
    {
        return 1;
    }
    else if (a.x < b.x)
    {
        return -1;
    }

    return a.y.CompareTo(b.y);    
});

foreach (var p in points)
{
    Console.WriteLine($"{p.x} {p.y}");
}
```

### 간결한 버전

```csharp
points.Sort((a, b) => a.x != b.x ? a.x.CompareTo(b.x) : a.y.CompareTo(b.y));
```
