# BOJ 2292 벌집

## 문제

벌집이 다음과 같이 육각형 모양으로 이루어져 있다. 중앙의 방 1부터 시작해서 이웃하는 방에 돌아가면서 1씩 증가하는 번호를 주소로 매길 수 있다. 

```
    19  20  21
  8   9  10  22
7   2   3  11  23
  6   1   4  12
18  5   4  13  24
  17  16  15  14
    36  35  34
```

숫자 N이 주어졌을 때, 벌집의 중앙 1에서 N번 방까지 최소 개수의 방을 지나서 갈 때 몇 개의 방을 지나가는지(시작과 끝을 포함하여) 구하는 프로그램을 작성하시오.

## 입력

- 첫째 줄: N (1 ≤ N ≤ 1,000,000,000)

## 출력

- 중앙의 1번 방에서 N번 방까지 최소 개수의 방을 지나서 가는 경로의 방 개수

## 접근

벌집 구조에서 각 층마다 방의 개수 패턴을 찾는 것이 핵심이다.

### 층별 방의 구조
- 중앙(0층): 1번 방 (1개)
- 1층: 2~7번 방 (6개)
- 2층: 8~19번 방 (12개)
- 3층: 20~37번 방 (18개)
- 4층: 38~61번 방 (24개)

### 패턴 발견
- 각 층마다 방의 개수가 6, 12, 18, 24... 즉 **6씩 증가**
- n층의 마지막 방 번호 = 1 + 3×n×(n+1)
- 또는 누적으로 계산: max_room += 6 × layer

### 알고리즘
1. N이 1이면 답은 1
2. 그 외에는 층을 하나씩 늘려가며 N이 속한 층을 찾음
3. 해당 층의 번호 + 1이 답 (중앙에서 시작하므로)

## 풀이 코드 (C#)

```csharp
using System;

int N = int.Parse(Console.ReadLine());

if (N == 1)
{
    Console.WriteLine(1);
}
else
{
    int layer = 1;
    int maxRoom = 1;
    
    while (maxRoom < N)
    {
        maxRoom += 6 * layer;
        layer++;
    }
    
    Console.WriteLine(layer);
}
```

## 메모

- 핵심 아이디어: 육각형 벌집의 층별 방 개수 패턴 찾기 (6의 배수)
- 수학적 규칙성을 찾는 문제
- while 루프로 간단하게 해결 가능
- 시간 복잡도: O(√N) - 층 수가 √N에 비례
