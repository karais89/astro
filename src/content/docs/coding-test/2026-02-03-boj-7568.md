# BOJ 7568 덩치

## 문제

N명의 사람들의 몸무게와 키가 주어졌을 때, 각 사람의 덩치 등수를 구하는 문제다.

**덩치 비교 규칙:**
- 사람 A의 (몸무게, 키)가 (x, y)이고 B가 (p, q)일 때
- x > p **그리고** y > q이면 A가 B보다 덩치가 크다
- 둘 중 하나라도 작거나 같으면 비교 불가능

**등수 계산:**
- 각 사람의 등수 = 자신보다 덩치가 큰 사람의 수 + 1
- 같은 등수가 여러 명 있을 수 있음 (예: 1등, 2등, 2등, 2등, 5등)

## 입력

- 첫째 줄: 전체 사람의 수 N (2 ≤ N ≤ 50)
- 다음 N개 줄: 각 사람의 몸무게 x와 키 y (10 ≤ x, y ≤ 200)

## 출력

- 입력에 나열된 순서대로 각 사람의 덩치 등수를 공백으로 구분하여 출력

## 접근

N이 최대 50이므로 O(N²) 브루트포스로 충분히 해결할 수 있다.

### 알고리즘

1. 각 사람마다 다른 모든 사람과 비교
2. 현재 사람보다 몸무게와 키가 **모두** 큰 사람의 수를 센다
3. 그 수에 1을 더해 등수로 출력

### 주의사항

- 정렬 불필요: 입력 순서대로 등수를 출력해야 함
- 비교 조건: 몸무게와 키 **둘 다** 커야 덩치가 크다고 판단
- 비교 불가능한 경우는 등수에 영향을 주지 않음

## 풀이 코드 (C#)

```csharp
public class Solution
{
    public struct Person
    {
        public int Weight;
        public int Height;
    }

    public static void Main(string[] args)
    {
        int n = int.Parse(Console.ReadLine()!);
        List<Person> persons = new List<Person>();
        for (int i = 0; i < n; i++)
        {
            var values = Array.ConvertAll(Console.ReadLine()!.Split(), int.Parse);
            Person p = new Person();
            p.Weight = values[0];
            p.Height = values[1];
            persons.Add(p);
        }

        for (int i = 0; i < n; i++)
        {
            Person current = persons[i];
            int rank = 1;
            for (int j = 0; j < n; j++)
            {
                if (i == j) continue;
                Person next = persons[j];
                if (current.Weight < next.Weight && current.Height < next.Height)
                {
                    rank++;
                }
            }
            Console.Write(rank + " ");
        }        
    }
}
```

## 메모

- 핵심 아이디어: N이 작으므로 모든 쌍을 비교하는 브루트포스 접근
- 두 조건(몸무게, 키)을 **모두** 만족해야만 덩치가 크다고 판단
- 정렬 없이 입력 순서대로 등수 출력
