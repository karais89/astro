## 1. 오늘 한 것

- GitHub Copilot 실제 사용량 체크 (결제 3일 경과 시점에서 약 10% 사용).
- 모델별 크레딧 소비량 분석 및 활용 전략 수립.
- 다양한 작업에 Copilot을 적용해보며 성능 평가.

## 2. 배운 점 / 느낀 점

- **모델별 사용 전략 수립**: GitHub Copilot은 월 300 크레딧 제공.
  - 간단한 작업(커밋 메시지, 간단한 수정): 무료 모델 사용 → 0 크레딧 (무제한).
  - 복잡한 작업: Sonnet 4.5 + Plan 모드 + Agent 모드 → 2 크레딧 (150회 실행 가능).
  - 고난도 작업: Opus 4.5 + Plan 모드 + Agent 모드 → 6 크레딧 (50회 실행 가능).
- **가성비 분석**: 10달러/월 가격 대비 합리적. 다만 무의식적으로 사용하면 크레딧이 부족할 수 있어 전략적 활용이 필요함.
- **크레딧 초과 시 과금**: 300 크레딧 초과 시 추가 요청당 **$0.04 USD** 과금. 단, 무료 모델(GPT-5 mini, GPT-4.1, GPT-4o)은 크레딧 소비 없이 무제한 사용 가능.
- **대용량 작업의 강점**: 40분 이상 작업도 1~2 크레딧으로 완료 가능. 토큰 소모량이 아닌 요청 기반 과금이라 대규모 작업에 무한 가성비.
- **전체 평가**: 오늘 많이 사용해본 결과, 다른 AI 에이전트에 비해 뒤쳐진다는 생각이 들지 않음.
