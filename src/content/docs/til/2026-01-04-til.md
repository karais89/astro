## 1. Google Antigravity 소개

Google에서 출시한 새로운 AI 에이전트 도구 Antigravity를 사용해보았다. 무료 계정으로 테스트했지만, 웹 개발 지식이 없어도 블로그를 수정할 수 있을 정도로 강력한 성능을 보여줬다.

현재 사용 가능한 모델:
- **Gemini 3 Pro**
- **Gemini 3 Flash**
- **Sonnet 4.5**
- **Opus 4.5**

## 2. Antigravity의 핵심: Planning 모드

가장 인상적이었던 기능은 Planning 모드다. 기본적으로 다음과 같이 동작한다:

1. **프롬프트 입력**: 사용자가 원하는 작업을 입력
2. **자동 플랜 생성**: AI가 해당 작업을 수행하기 위한 단계별 플랜 생성
3. **플랜 리뷰 및 수정**: 사용자가 플랜을 코드 리뷰하듯 세부 내용을 수정 가능
4. **실행**: 승인된 플랜에 따라 작업 진행

다른 도구와 달리 **실행 전에 플래닝을 먼저 세운다는 점**이 특징적이었다. 불필요한 시도를 줄이고 정확도를 높이는 데 큰 도움이 되는 것 같다.

## 3. AI 에이전트를 활용한 ASTRO 블로그 수정 경험

웹 개발 지식이 전혀 없지만, ASTRO 블로그 내용을 수정하는 과정에서 Antigravity를 적극적으로 활용했다. 
에이전트가 ASTRO 프로젝트 구조를 이해하고, 수정이 필요한 부분을 식별하며, 코드를 수정하고 웹 브라우저를 통해 자동으로 디버깅까지 진행 하는 모습이 인상적이었다. 내가 프롬프트로 원하는 기능을 설명하면, AI가 알아서 필요한 작업을 파악하고 실행했다.

## 4. 웹 개발 지식 없이 겪은 자동화 경험

특히 **자동 디버깅** 기능이 놀라웠다. 코드 수정 후 발생하는 오류를 스스로 파악하고 수정하는 과정을 반복하면서, 내가 전혀 몰라도 되는 부분까지 처리했다.

웹 프레임워크나 스타일링에 대한 이해가 없어도, 자연어로 원하는 기능을 설명하면 구현해주는 것이 큰 장점이었다.

## 5. 무료 계정의 한계점과 장점

### 장점
- 무료임에도 상당히 강력한 성능
- Planning 모드를 통해 작업 효율화
- 다양한 모델 선택 가능성
- 웹 개발 비전문자도 사용 가능한 UX

### 한계점
- **할당량이 생각보다 작음**: 빠르게 소진될 수 있음
- **할당량 리셋 주기가 7일**: 연속 사용에는 불편할 수 있음

## 6. 결론

Antigravity는 무료 계정이라는 제약이 있지만, Planning 모드와 자동 디버깅 기능을 통해 상당히 인상적인 성능을 보여줬다. 웹 개발 지식이 없어도 블로그 수정이 가능했던 경험은 AI 에이전트의 실용성을 다시금 확인해주었다.

할당량 리셋이 7일 후라는 점과 한정된 사용량은 단점이지만, 무료 플랜을 감안하면 충분히 만족스러운 경험이었다. 현재는 프리뷰 버전이지만, 향후 유료 플랜이 생긴다면 사용도 고려해볼 만한 가치가 있다고 생각한다.
