## 2. 배운 점 / 느낀 점

### GitHub Copilot CLI vs VSCode 확장 버전

**초기 인식의 변화**
- 처음에는 CLI 버전을 왜 사용해야 하는지 명확한 이유를 느끼지 못했음
- 지속적인 빠른 업데이트로 인해 점차적으로 사용 가치가 증가 (현재 프리뷰 버전)

**장점**

1. **Plan Mode의 명확성과 편의성**
   - VSCode의 Plan 모드보다 더 명확한 구조 제공
   - 별도의 파일로 계획을 생성
   - `Ctrl + Y` 등의 키로 Plan 모드 내 수정이 즉시 가능
   - VSCode에서는 별도 파일 생성이 없을 경우 다음 Agent에서 모든 내용을 수정해야 하는 번거로움 있음

2. **GPT 모델의 추론 설정**
   - Reasoning Effort 설정 가능
   - 단, 최고 수준 모델까지 선택은 아직 지원되지 않음

3. **컨텍스트 관리**
   - 현재 사용 중인 컨텍스트 확인 가능 (/context)
   - 컨텍스트 관리가 용이

**단점**

1. **Windows 환경에서의 문제**
   - PowerShell 관련 명령이 제대로 실행되지 않는 이슈 존재
